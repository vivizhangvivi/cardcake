# CardCake 小蛋糕笔记 🍰

A tiny card-making tool for writing beautiful notes and sharing them
on social platform. 
Pick a flavor, type your thought, export your card.

## ✨ Try it live
👉 [https://vivizhangvivi.github.io/cardcake]

## What it does
- 8 color themes (Vanilla, Matcha, Raspberry...)
- 6 layout styles (Breathe, Dialog, Sandwich...)
- Font & spacing tuning
- PNG export
- Markdown text output
- Save your favorite style combos as presets

## How to use
Just open the link above — no install needed.
To run locally: download index.html and open it in any browser.
